GBIF

GBIF (Global Biodiversity Information Facility https://www.gbif.org/) is an open access biodiversity database. This is an intergovermental database started in 2001, and is currently contributed by 97 countries with over 1500 publishers (mainly universities, museums, scientific institutions).

#rgbif rgbif is an R package to communicate with GBIF, that let you search for information of any taxonomic group of your interest, and count or download their occurence records.

Installation
r library(rgbif)
## Warning: package 'rgbif' was built under R version 3.5.3

Count occurence records

occ_count() will simply let you count occurence records in GBIF with various arguments.

e.g. 

# Count by observation types.
occ_count(basisOfRecord='HUMAN_OBSERVATION')
## [1] 1108123709
occ_count(basisOfRecord='MACHINE_OBSERVATION')
## [1] 11917523
occ_count(basisOfRecord='LITERATURE')
## [1] 219987
occ_count(basisOfRecord='LIVING_SPECIMEN')
## [1] 1637974
occ_count(basisOfRecord='FOSSIL_SPECIMEN')
## [1] 9045358
# Count records that have lat/long data.
occ_count(georeferenced = TRUE)
## [1] 1261155991
# Count records in a specific year.
occ_count(year = 2012)
## [1] 58269042
# Count data for a specific taxa.
# Taxa: genus *Ambrosia* (ragweed)
# Taxon key: 3110576
occ_count(taxonKey=3110576)
## [1] 71079
# Count data for a country.
# Country: United States
# Country code: US
occ_count(country = "US")
## [1] 462028238

You can combine multiple data conditions in occ_count arguments. e.g. Count records of ragweed in USA with lat/long data.

# Observation type: Human Observation
# Country: United States (country code = US)
# Taxon: *Ambrosia* (taxonKey= 3110576)
# Has long/lat data
occ_count(basisOfRecord='HUMAN_OBSERVATION', country = "US", taxonKey=3110588, georeferenced = TRUE)
## [1] 3240

Obtain occurence records >500

In order to download a large occurence records at once, you need a GBIF account. Store your user name, passward, and email address as your environmental variables (.Rprofile, .Renviron) or simply set them as R objects.

user <- "namiko"
pwd <- "735seibutsu"
email <- "nm0076@mix.wvu.edu"
  1. Make a download request First, you need to make a download request to the GBIF by using occ_download() and let the database compile the data for you. For downloading format, you can choose one from DWCA(Darwin Core Archive), SIMPLE_CSV, and SPECIES_LIST.

e.g. Taxon: genus Ambrosia (key = 3110576) Country: United States (code = US) Download format: Darwin Core Archive

dl_Ambrosia_USA <- occ_download('taxonKey = 3110576', 'country = US', format = "DWCA", user = user, pwd = pwd, email = email)
dl_Ambrosia_USA
## <<gbif download>>
##   Username: namiko
##   E-mail: nm0076@mix.wvu.edu
##   Format: DWCA
##   Download key: 0010354-191105090559680

Now, you have to wait for a while, from a few minutes up to 3 hours, depending on the amount of data you requested. Check the status in https://www.gbif.org/user/download

  1. Fetch the data from database By using the download key provided by occ_download(), you can now download the large occurence data from the GBIF to your workspace or save them to your computer as a ZIP file.

e.g. Import data into environment

# Download key
dl_Ambrosia_USA <- "0009448-191105090559680"

# occ_download_import(): import data into your workspace
Ambrosia_USA <- occ_download_import(key = dl_Ambrosia_USA)

Get citation infomation of the data.

gbif_citation() will provide a list of all citations for the datasets that contribute to your occurence records. The input can be the result of occ_search(), occ_download_get(), etc.

# Country code
usa_code <- isocodes[grep("^United States$", isocodes$name), "code"]
# Occurence records
occurrence_data <- occ_search(scientificName = "Ambrosia intergradiens", country=usa_code, hasCoordinate = TRUE, return='data')

Plot occurence data on map

rgbif has a mapping function called map_fetch() but it’s still under development. Instead, ggplot works well with the occurence records obtained from GBIF.

library(ggplot2)
## Warning: package 'ggplot2' was built under R version 3.5.3

e.g. Ragweed (genus Ambrosia) distribution in USA.

# Download data
dl_Ambrosia_USA <- occ_download('taxonKey = 3110576', 'country = US', format = "DWCA", user = user, pwd = pwd, email = email)
Ambrosia_USA <- occ_download_import(key = dl_Ambrosia_USA)
# Plot with ggplot2
usa <- map_data("usa")
ggplot(usa, aes(long, lat)) +
ggtitle("Species distribution of ragweed (Genus Ambrosia)") +
geom_polygon(aes(group = group), fill = "white", 
              color = "black", size = .5) +
geom_jitter(data = Ambrosia_USA, aes(decimalLongitude, decimalLatitude, color = specificEpithet), alpha=1, size = 1, show.legend = TRUE) +
coord_map(xlim = c(-130, -60), ylim = c(20, 50)) +
theme(legend.position="bottom", legend.box = "horizontal", legend.text  = element_text(size = 7), legend.key.size = unit(0.1, "lines"))
## Warning: Removed 10433 rows containing missing values (geom_point).
